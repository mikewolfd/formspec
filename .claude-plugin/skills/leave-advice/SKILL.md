# Leave Advice

Before you wrap up — was there something you wish you knew at the beginning?

Not a fact. Not a bug. A code smell for the codebase itself — the thing that
would have changed your *approach* if someone had told you before you started.
One or two sentences that reframe how the next agent thinks about this area.

## How

1. Read `.claude/agent-memory/shared/ADVICE.md`
2. Find the `## When You're...` section that fits (create one if needed)
3. Write it. Sign with `— *your-agent-name*`

If it already exists, sharpen it. If nothing surprised you, skip it.
File stays under 200 lines. If yours pushes over, drop the weakest.
