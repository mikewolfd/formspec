AI-native

Built for the
AI era.

Every Formspec form is a structured JSON document. That means any AI model can read it, fill it, or explain it — without scraping HTML or guessing field names. It speaks AI fluently, by design.

Structured output LLM form generation AI pre-fill GPT-4 / Claude / Gemini JSON Schema constrained Natural language errors
1
AI generates the form
Describe what you need in plain English. The AI outputs a valid Formspec definition with fields, rules, and validation — constrained by the schema so it's always correct.
2
User fills it in (or AI does)
Forms run live in the browser with real-time conditional logic. Or pass existing data to an AI and let it pre-fill the response — the same JSON schema validates either way.
3
AI explains what's wrong
Validation errors are structured JSON — not cryptic codes. Feed them back to any LLM and get plain-English explanations your users can actually act on.
4
Server re-verifies everything
The same rules that ran in the browser run again on your server. No trust gap. No duplicate logic to maintain.
